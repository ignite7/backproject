What is this?

This is an API open source project, the main aim of this API is allow to the user search between several services in one page, you can search between movies, series, games and music for now.

The API works with others database API's for get the information that maybe you are watching now,. If you are more interesting this is an open source project, you can check out on github.

How can I use it?

The usage it's simple if you want make a search make it in the search bar, it's important to know if you are a visitor you can only make 3 searches per hour becuase this is only an instance of our API, if you want to use the API for develop porpuses signup and get your keys to work with the API and of course here you have the documentation.
