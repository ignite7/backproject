All you need to know to get started with the back project

This API open source project was made with Django REST framework, which is a framework written in python and defines itself "Powerful and flexible toolkit for building Web APIs."

The resources are obtained from IMDb for the movies and series, IGDB for the games and Spotify for the music.

Authentication and versioning works in the headers, for example:

Key Value
Authorization authorization key
Secret secret key
Accept application/json; version=v1
